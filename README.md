# **Madrid House Price Analysis**

## **Project Description**
### **This project analyzes house prices in Madrid using real estate data to uncover trends, patterns, and factors influencing property prices. The analysis includes data exploration, preprocessing, visualization, and predictive modeling .**

## **Objectives**
- ### **Exploratory Data Analysis (EDA): Investigate the distribution of house prices and identify key features (e.g., location, size, number of bedrooms) affecting prices.**

- ### **Data Preprocessing: Clean the dataset by handling missing values, and encoding categorical variables.**

- ### **Visualization: Create visualizations (e.g., histograms, scatter plots, heatmaps) to illustrate price trends and correlations.**

- ### **Predictive Modeling: Build machine learning models to predict house prices based on available features.**


## **Tools**

### **Programming Language: Python**
### **Libraries:**
- ### **Data Manipulation: pandas, numpy**
- ### **scikit-learn**
- ### **statsmodels**
- ### **Visualization: matplotlib, seaborn**
